# Публикация комиксов

Программа публикует случайный комикс xkcd в Telegram-канал.

## Установка

### Получение ключей API

Для работы программы требуются следующие ключи:

#### Токен Telegram-бота
1. Напишите @BotFather в Telegram
2. Отправьте команду /newbot и следуйте инструкциям
3. Получите токен вашего бота (выглядит как 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz)
4. Создайте канал и добавьте вашего бота как администратора

#### ID чата Telegram
1. После добавления бота в канал отправьте любое сообщение
2. Получите ID чата через @userinfobot

Создайте файл `.env` и добавьте туда токен бота и ID чата. Пример:
```
TELEGRAM_BOT_TOKEN=[Токен бота]
TELEGRAM_CHAT_ID=[Имя Telegram-канала, в который отправляются комиксы]
```

Python3 должен быть уже установлен.
Затем используйте pip (или pip3, если есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

## Запуск

### Примеры использования скриптов

#### Публикация случайного комикса
```
# Публикует случайный комикс xkcd в Telegram-канал
python main.py
```
